# Galaxy-Defenders
Criado com muita dedicação por alunos do curso de Ciência da Computação da UNIMA Afya Maceió, este jogo é uma homenagem divertida ao clássico Space Invaders. Nele, você assume o controle de uma nave espacial e precisa defender o planeta contra ondas de inimigos que vêm de todos os lados. 

Link que da acesso ao jogo 'https://galaxydefendersofc.netlify.app'

O codigo base do jogo foi feito em Java script, Html, Css, JSON

API utilizada 'https://developers.google.com/fonts/docs/css2?hl=pt-br'

Link de demonstração do jogo = https://youtu.be/HJTc6wlDlzE
